The executable file is named "Snoopy Adventure in Uniqlo.apk" in the root folder, it should be installed on an Android device with Android 4.0+

Mouse control can be enabled in editor play mode, however the script for mouse control is commented out.
To activate mouse control, go to the Assets/Scripts/SwipeHandler. Inside void Update(), remove the comment tag for the first fragment of code.
